# AI Coding Assistant

An AI-powered coding assistant that analyzes GitHub repositories, explains code, reviews pull requests, detects bugs, and generates documentation using LLMs and Retrieval-Augmented Generation (RAG).

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- Qdrant
- LangGraph
- Next.js
- Docker
